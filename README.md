 # coder Flex App

 ### Instalacion
 1. Clone el Repositorio
 2. Hacer el comando`cd proyereact ` para moverse a la carpeta raiz del proyecto
 3. Instale dependencia con el comand `npm install`
 4. Para levantar el proyecto tipear y levantarlo local `npm run dev `
